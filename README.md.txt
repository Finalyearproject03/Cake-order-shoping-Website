About This Project
The Cake Shop Management System major project is a website that is based on managing the order and selling the Cake online. A system that allows users to check for various cakes available at the online store order custom cake with flavor toppings names etc. and purchase online. The customers can order instantaneously anywhere around the area, anywhere they want. It will make a person’s life easier in a dramatic way. If a person wishes to buy a cake, that person just has to go to this system, select the cake, provide some important information, and make the payment.
